# Resume-Screening
AI-Powered Resume Screening System using NLP and Machine Learning
Overview

The AI-Powered Resume Screening System is an intelligent recruitment automation solution that leverages Natural Language Processing (NLP), Machine Learning, and Text Analytics to streamline candidate shortlisting and resume classification. The system automatically analyzes resume content, extracts meaningful textual patterns, and predicts the most relevant job category, significantly reducing manual screening effort.

This project demonstrates the practical application of NLP pipelines, feature engineering, text vectorization, and supervised learning algorithms in Human Resource Analytics and Talent Acquisition.

Problem Statement

Organizations receive thousands of resumes for multiple job openings, making manual screening inefficient, time-consuming, and difficult to scale.

The objective of this project is to:

Automate candidate screening workflows
Classify resumes into relevant job domains
Reduce recruiter workload
Improve hiring efficiency
Enable data-driven recruitment decisions
Build an intelligent candidate recommendation pipeline

Feature Engineering
TF-IDF Vectorization

Term Frequency–Inverse Document Frequency (TF-IDF) was used to transform unstructured textual resumes into numerical feature vectors.

Benefits:

Captures contextual importance of skills
Reduces impact of frequently occurring terms
Generates high-dimensional sparse representations
Improves classification performance
